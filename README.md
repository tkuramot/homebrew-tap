# homebrew-tap

Homebrew tap for [@tkuramot](https://github.com/tkuramot)'s tools.

## Usage

Add the tap:

```sh
brew tap tkuramot/tap
```

Then install any of the casks below:

```sh
brew install --cask <name>
```

Or in one step:

```sh
brew install --cask tkuramot/tap/<name>
```

## Available Casks

| Name | Description | Repository |
| --- | --- | --- |
| `fumi` | Browser × host machine integration utility | [tkuramot/fumi](https://github.com/tkuramot/fumi) |
| `muxrun` | Launch and manage multiple applications in groups using tmux (requires `tmux`) | [tkuramot/muxrun](https://github.com/tkuramot/muxrun) |

## Notes

- Cask files in `Casks/` are auto-generated by [GoReleaser](https://goreleaser.com/) on each upstream release — do not edit them by hand.
- Issues with a specific tool should be filed on its own repository.
